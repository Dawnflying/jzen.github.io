## 💡 快速提示

### 如何写第一篇文章？

```
1. 点击"写文章"
2. 填写标题（例如："我的博客启航"）
3. 选择分类（中国哲学 或 中国历史）
4. 添加标签（按回车键添加）
5. 在编辑器中写内容
6. 点击"保存文章"
```

### 如何自定义配色？

编辑 `src/index.css`，修改 CSS 变量：

```css
:root {
  --primary-red: #c8102e;    /* 主色 */
  --primary-gold: #d4af37;   /* 强调色 */
  /* ...更多颜色... */
}
```

### 如何添加自己的音乐？

1. 将音乐文件（.mp3）放到 `public` 文件夹
2. 编辑 `src/components/MusicPlayer.jsx`
3. 修改 `musicUrl` 为你的文件路径
