# 禅境博客 - 使用指南

欢迎使用禅境博客！本指南将帮助您快速上手。

## 🚀 快速开始

### 安装和运行

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 在浏览器中打开 http://localhost:3000
```

## 📝 创建第一篇文章

### 步骤 1：进入编辑器

1. 点击顶部导航栏的"写文章"按钮
2. 或访问 `/editor` 路由

### 步骤 2：填写基本信息

- **标题**: 输入文章标题（最多100字符）
- **分类**: 选择"中国哲学"或"中国历史"
- **作者**: 输入作者名称（默认为"禅境居士"）
- **标签**: 输入标签后按回车键添加，最多可添加多个标签

### 步骤 3：编写内容

使用富文本编辑器撰写文章：

#### 工具栏功能：

- **标题**: 选择 H1、H2、H3 标题级别
- **格式**: 加粗、斜体、下划线、删除线
- **颜色**: 文字颜色和背景色
- **列表**: 有序列表、无序列表
- **引用**: 创建引用块
- **代码**: 添加代码块
- **对齐**: 文本对齐方式
- **链接**: 插入超链接

#### 编写技巧：

**标题使用**:
```
# 一级标题
## 二级标题
### 三级标题
```

**引用格式**:
> 道可道，非常道
> 名可名，非常名

**列表格式**:
- 项目一
- 项目二
  - 子项目

### 步骤 4：预览

1. 点击"预览"按钮查看文章效果
2. 再次点击返回编辑模式

### 步骤 5：保存发布

1. 点击"保存文章"按钮
2. 系统会自动保存并跳转到文章详情页

## 📖 阅读和管理文章

### 首页浏览

- 查看所有已发布的文章
- 使用分类筛选器过滤文章
- 点击文章卡片进入详情页

### 文章详情页

**阅读功能**:
- 查看完整文章内容
- 浏览文章元信息（日期、浏览量等）

**互动功能**:
- 点击❤️按钮为文章点赞
- 在评论区发表评论

**管理功能**:
- 点击"编辑"修改文章
- 点击"删除"删除文章（需确认）

## 💬 评论系统

### 发表评论

1. 在文章详情页滚动到评论区
2. 输入您的名字（必填）
3. 输入评论内容（必填，最多500字符）
4. 点击"发表评论"

### 查看评论

- 评论按时间倒序显示（最新的在上面）
- 显示评论者名字和发表时间
- 支持相对时间显示（如"5分钟前"）

## ❤️ 点赞功能

- 每篇文章只能点赞一次
- 点赞后按钮变红
- 再次点击取消点赞
- 点赞数实时更新

## 🎵 音乐控制

右下角的音乐播放器提供以下功能：

- **播放/暂停**: 点击播放按钮
- **静音/取消静音**: 点击音量按钮
- **音量调节**: 拖动音量滑块

**更换音乐**:

编辑 `src/components/MusicPlayer.jsx`：

```javascript
const musicUrl = 'your-music-file.mp3'
```

建议使用：
- 古筝音乐
- 古琴音乐
- 中国风轻音乐

## ❄️ 雪花效果

- 背景自动显示飘雪动画
- 可以通过修改 `src/components/SnowEffect.jsx` 调整效果
- 调整雪花数量、大小、速度等参数

## 🎨 自定义样式

### 修改主题色

编辑 `src/index.css`：

```css
:root {
  --primary-red: #c8102e;      /* 主色调 */
  --primary-gold: #d4af37;     /* 强调色 */
  --dark-ink: #2c2c2c;         /* 深色文字 */
  --light-ink: #5a5a5a;        /* 浅色文字 */
  --paper-white: #faf9f6;      /* 背景色 */
  --bamboo-green: #5f7a61;     /* 辅助色 */
  --silk-beige: #e8dcc4;       /* 边框色 */
}
```

### 修改字体

在 `index.html` 中更改 Google Fonts 链接，然后在 CSS 中应用。

## 💾 数据管理

### 数据存储位置

所有数据存储在浏览器的 LocalStorage 中：

- **文章数据**: `zen_blog_posts`
- **评论数据**: `zen_blog_comments`
- **点赞数据**: `zen_blog_likes`

### 导出数据

打开浏览器开发者工具（F12），在 Console 中运行：

```javascript
// 导出所有文章
const posts = localStorage.getItem('zen_blog_posts')
console.log(posts)

// 导出评论
const comments = localStorage.getItem('zen_blog_comments')
console.log(comments)

// 导出点赞数据
const likes = localStorage.getItem('zen_blog_likes')
console.log(likes)
```

### 导入数据

```javascript
// 导入文章
localStorage.setItem('zen_blog_posts', yourPostsData)

// 刷新页面
location.reload()
```

### 清除所有数据

```javascript
localStorage.removeItem('zen_blog_posts')
localStorage.removeItem('zen_blog_comments')
localStorage.removeItem('zen_blog_likes')
location.reload()
```

## 🔧 故障排除

### 问题：页面无法加载

**解决方案**:
1. 检查是否运行 `npm install`
2. 确认 Node.js 版本（需要 v16+）
3. 清除浏览器缓存
4. 查看控制台错误信息

### 问题：文章保存失败

**解决方案**:
1. 检查标题和内容是否已填写
2. 检查浏览器是否支持 LocalStorage
3. 检查是否达到存储限制（约 5-10MB）

### 问题：音乐无法播放

**解决方案**:
1. 检查音乐文件 URL 是否有效
2. 检查浏览器是否允许自动播放
3. 尝试手动点击播放按钮
4. 检查音频格式兼容性

### 问题：雪花效果卡顿

**解决方案**:
1. 减少雪花数量
2. 在低性能设备上考虑禁用
3. 检查其他占用资源的程序
