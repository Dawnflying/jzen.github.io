# 主题切换使用指南

## 🌓 主题模式

博客支持两种主题模式：

### ☀️ 明色调 - 寒江雪景
- 清爽明亮的冰蓝色调
- 适合白天阅读
- 高对比度，易于阅读
- 模拟冬日雪景的明亮感

### 🌙 暗色调 - 寒夜雪境
- 深沉宁静的夜空色调
- 适合夜间阅读
- 护眼舒适，降低眼部疲劳
- 模拟雪夜的幽静氛围

## 🎨 配色方案

### 明色调（默认）
```css
背景渐变：#d4dfe8 → #e8eef3 → #f0f4f8
主题色：冰蓝 #a8c8d8
强调色：冬日蓝 #6a8caf
文字色：深墨 #2d3e50
卡片背景：白色半透明
```

### 暗色调
```css
背景渐变：#0f1419 → #1a2332 → #1e2937
主题色：月光蓝 #7ba3c0
强调色：寒夜蓝 #5a7fa3
文字色：霜白 #e8ecf0
卡片背景：深蓝灰半透明
```

## 💡 如何切换主题

### 方法一：使用主题切换按钮
1. 在页面左下角找到主题切换器
2. 点击切换按钮
3. 明色模式显示🌙图标和"夜"字
4. 暗色模式显示☀️图标和"昼"字

### 方法二：通过代码
```javascript
import { toggleTheme } from './utils/theme'

// 切换主题
toggleTheme()
```

## 🎯 主题特性

### 自动保存
- 主题选择会自动保存到 LocalStorage
- 下次访问时自动恢复上次选择的主题

### 平滑过渡
- 所有颜色变化都有 0.3-0.5 秒的过渡动画
- 切换体验流畅自然

### 全局适配
- 所有页面都支持主题切换
- 所有组件都自动适配当前主题
- 包括雪花颜色也会随主题变化

## 🌟 主题适配的元素

### ✅ 已适配元素
- 整体背景
- 导航栏
- 所有卡片
- 文章内容
- 评论区
- 编辑器（含工具栏）
- 按钮
- 输入框
- 模态框
- 音乐播放器
- 主题切换器本身
- 雪花效果

### 特殊处理
1. **雪花颜色**
   - 明色：白色偏蓝 (230, 240, 248)
   - 暗色：亮白偏蓝 (200, 220, 235)

2. **代码块**
   - 明色：深色背景 + 浅色文字
   - 暗色：GitHub 暗色风格

3. **竹简纹理**
   - 两种主题下都保持竹简纹理
   - 透明度自动调整

## 🛠️ 开发者自定义

### 修改主题颜色

编辑 `src/index.css`：

```css
/* 修改明色主题 */
:root {
  --ice-blue: #yourColor;
  /* ... 其他颜色 */
}

/* 修改暗色主题 */
[data-theme="dark"] {
  --ice-blue: #yourDarkColor;
  /* ... 其他颜色 */
}
```

### 添加新的主题变量

```css
:root {
  --custom-color: #yourColor;
}

[data-theme="dark"] {
  --custom-color: #yourDarkColor;
}
```

### 在组件中使用

```css
.your-element {
  color: var(--custom-color);
  transition: color 0.3s ease; /* 添加过渡效果 */
}
```

## 💡 使用建议

### 何时使用明色调
- ✅ 白天阅读
- ✅ 光线充足的环境
- ✅ 需要高对比度时
- ✅ 打印或截图时

### 何时使用暗色调
- ✅ 夜间阅读
- ✅ 光线昏暗的环境
- ✅ 长时间阅读
- ✅ 减少眼部疲劳

## 🎨 设计理念

### 明色调 - "寒江雪景"
灵感来自柳宗元《江雪》的白天场景：
- 千山白雪，万径无人
- 清冷明亮，天地苍茫
- 孤舟渔翁，独钓江心

### 暗色调 - "寒夜雪境"
延伸为夜间的意境：
- 夜幕低垂，雪映寒星
- 深沉宁静，月色清寒
- 孤灯蓑笠，夜钓江雪

## 🔧 技术细节

### 实现方式
- 使用 CSS 变量（Custom Properties）
- `data-theme` 属性切换
- LocalStorage 持久化
- React Context（可选）

### 浏览器兼容性
- ✅ Chrome 88+
- ✅ Firefox 85+
- ✅ Safari 14+
- ✅ Edge 88+

### 性能优化
- 使用 CSS 变量避免重复样式
- 过渡动画控制在 0.5s 以内
- 按需加载主题相关资源

## 📝 注意事项

1. **图片适配**：如果添加图片，建议提供明暗两个版本
2. **对比度**：确保文字在两种主题下都有足够对比度
3. **测试**：新增组件时记得测试两种主题
4. **过渡**：所有颜色相关属性都应该添加 `transition`
